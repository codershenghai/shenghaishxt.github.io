首先给出最短路径算法的几种类型：

  • Point-to-Point Shortest Path，点到点最短路径：给定起点和终点，求出起点到终点的最短路径。
  • Single Source Shortest Paths，单源最短路径：给定起点，求出起点到图上每一点的最短路径。
  • All Pairs Shortest Paths，多源最短路径：求出图上所有两点之间的最短路径。
